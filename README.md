#一套优雅的IM实现方案：UI篇

1. 聊天相关的所有ui:
    - 自定义键盘：可实现文本，emoij表情，自定义gif表情发送
2. 可模拟发送以下消息类型：
      -  文本消息
      -  音频消息
      -  视频消息
      -  照片消息
      -  位置消息
 
3. 可模拟接收消息：接收消息类型与发送一致。
  
4. 定制了一系列接口：
     - im底层的接口：接入webSock，或xmpp。
     - 媒体消息类型处理的接口：转码或者其他的展现形式
  
5. 设计了消息模型。
